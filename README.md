# My dotfiles

## Overview
This repo is my dotfiles.

## My dotfiles
```
nvim/init.vim         => ~/.config/nvim/init.vim  
vim/.vimrc            => ~/.vimrc  
zsh/.zshrc            => ~/.zshrc  
tmux/.tmux.conf       => ~/.tmux.conf
tmux/.tmux.conf.local => ~/.tmux.conf.local
```

and use [uradotfiles](https://github.com/NoahOrberg/uradotfiles) (priv repository)

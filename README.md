# My dotfiles

## Overview
This repo is my dotfiles.

## My dotfiles
```
~/.config/nvim/init.vim  
~/.vimrc  
~/.zshrc  
~/.tmux.conf
```

## Using this repo
**!!! Be careful: Your dotfiles will be overwritten**
``` bash
$ chmod u+x ./install.sh
$ ./install.sh
```

and use [uradotfiles](https://github.com/NoahOrberg/uradotfiles) (priv repository)
